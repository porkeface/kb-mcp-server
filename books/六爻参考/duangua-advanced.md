# 进阶断卦规则 — 《增删卜易》

## 一、进神与退神

### 1.1 进神

动爻化出的变爻地支，在十二地支中**向前进**（按寅卯辰巳午未申酉戌亥子丑的五行旺衰方向），即为进神。

**进神条件**（同一五行内地支前进）：

| 动爻地支 | 化进地支 | 五行 |
|---|---|---|
| 寅 | 化卯 | 木进 |
| 巳 | 化午 | 火进 |
| 申 | 化酉 | 金进 |
| 亥 | 化子 | 水进 |
| 辰 | 化丑 | 土进（一说丑化辰） |
| 未 | 化戌 | 土进（一说戌化未） |

**进神含义**: 用神化进神，主事情向好的方向发展，越来越旺。忌神化进神则凶上加凶。

### 1.2 退神

动爻化出的变爻地支，在同一五行内**向后退**，即为退神。

| 动爻地支 | 化退地支 | 五行 |
|---|---|---|
| 卯 | 化寅 | 木退 |
| 午 | 化巳 | 火退 |
| 酉 | 化申 | 金退 |
| 子 | 化亥 | 水退 |
| 丑 | 化辰 | 土退（一说辰化丑） |
| 戌 | 化未 | 土退（一说未化戌） |

**退神含义**: 用神化退神，主事情退缩、衰减。忌神化退神则忌神退去，反为好事。

## 二、六合卦与六冲卦

### 2.1 六冲卦（整卦层面）

六冲卦是指上下卦对应爻位互冲的卦。六冲卦主散、主快、主不持久。

**六冲卦列表**:
| 卦名 | 上卦 | 下卦 |
|---|---|---|
| 乾为天 | 乾 | 乾 |
| 兑为泽 | 兑 | 兑 |
| 离为火 | 离 | 离 |
| 震为雷 | 震 | 震 |
| 巽为风 | 巽 | 巽 |
| 坎为水 | 坎 | 坎 |
| 艮为山 | 艮 | 艮 |
| 坤为地 | 坤 | 坤 |
| 天雷无妄 | 乾 | 震 |
| 雷天大壮 | 震 | 乾 |
| 风山渐 | 巽 | 艮 |
| 山风蛊 | 艮 | 巽 |
| 水火既济 | 坎 | 离 |
| 火水未济 | 离 | 坎 |
| 泽地萃 | 兑 | 坤 |
| 地泽临 | 坤 | 兑 |

判断方法：初爻与四爻冲、二爻与五爻冲、三爻与上爻冲，三对皆冲即为六冲卦。

**六冲卦断语**:
- 凡占事遇六冲卦，主事不成、不久、散
- 占婚姻遇六冲，不成或离散
- 占病遇六冲，病情反复或不治
- 但占仇怨、散伙等事遇六冲反吉

### 2.2 六合卦（整卦层面）

六合卦是指上下卦对应爻位互合的卦。六合卦主合、主成、主持久。

**六合卦列表**:
| 卦名 | 上卦 | 下卦 |
|---|---|---|
| 地天泰 | 坤 | 乾 |
| 天地否 | 乾 | 坤 |
| 风雷益 | 巽 | 震 |
| 雷风恒 | 震 | 巽 |
| 水山蹇 | 坎 | 艮 |
| 山水蒙 | 艮 | 坎 |
| 火泽睽 | 离 | 兑 |
| 泽火革 | 兑 | 离 |

判断方法：初爻与四爻合、二爻与五爻合、三爻与上爻合，三对皆合即为六合卦。

**六合卦断语**:
- 凡占事遇六合卦，主事可成、持久、和合
- 占婚姻遇六合，成，且美满
- 但占散伙、脱身等事遇六合反不利

### 2.3 变卦合冲

- 本卦六冲变六合：先散后聚，先凶后吉
- 本卦六合变六冲：先合后散，先吉后凶
- 本卦六冲变六冲：彻底散，凡事不成
- 本卦六合变六合：事情顺利，持久

## 三、反吟与伏吟

### 3.1 反吟

**反吟**指动爻化出的变爻与动爻形成六冲关系（地支相冲），或整卦层面本卦与变卦形成六冲。

| 类型 | 说明 | 含义 |
|---|---|---|
| 爻反吟 | 动爻地支与变爻地支六冲 | 反复不定，事情来回折腾 |
| 卦反吟 | 本卦与变卦互为六冲 | 大反复，事情翻覆 |

**反吟断语**: 反吟主反复、忧虑、不安。用神反吟主事情反复，世爻反吟主自身不安。

### 3.2 伏吟

**伏吟**指动爻化出的变爻与动爻地支相同（如子动化子），或整卦不动。

| 类型 | 说明 | 含义 |
|---|---|---|
| 爻伏吟 | 动爻地支与变爻地支相同 | 郁闷不展，呻吟之象 |
| 卦伏吟 | 全卦无动爻（六爻皆静） | 事情停滞，无进展 |

**伏吟断语**: 伏吟主愁闷、停滞、呻吟。事情难以推进，需等待时机。

## 四、飞神与伏神

### 4.1 何时需要找伏神

当卦中六爻没有所需的用神（某一六亲完全不出现在卦中），则需要在**本宫八纯卦**（本宫卦）中寻找伏神。

### 4.2 寻找伏神的方法

1. 确定本卦所属宫
2. 找到该宫的本宫卦（八纯卦，如乾宫找乾为天）
3. 在本宫卦中找到缺失的那个六亲所在的爻位
4. 该爻即为**伏神**，伏在本卦相同爻位之下
5. 本卦该爻位的爻称为**飞神**

### 4.3 伏神能否发挥作用

| 条件 | 效果 |
|---|---|
| 伏神得日月生扶 | 伏神有力，可用 |
| 伏神得飞神生扶 | 伏神有力，飞神助伏神出 |
| 伏神被飞神克制 | 伏神被压制，难以发挥 |
| 伏神旺相 | 虽伏但有力，逢冲飞之日或合处伏之日可出 |
| 伏神休囚 | 无力，难以发挥作用 |
| 伏神被飞神合住 | 被牢牢压住，更难出 |

### 4.4 伏神出现的时机

- 冲飞之日（冲开飞神，伏神得出）
- 合处伏神之日（合起伏神）
- 伏神值日之日
- 伏神逢旺之月

## 五、独发与乱动

### 5.1 独发（一爻独动）

卦中只有一个动爻时：
- 该动爻为主要断卦依据
- 看该动爻与用神的生克关系
- 看该动爻的变爻（回头生克）

### 5.2 两爻动

- 以与用神关系更直接的动爻为主
- 两动爻之间也有生克作用

### 5.3 多爻动（三爻以上）

- 卦中动爻过多，局势复杂
- 以用神本身的状态为主
- 看哪个动爻对用神影响最大
- 变卦的意义加重

### 5.4 六爻全动

- 以变卦为主断
- 本卦为起因，变卦为结果
- 乾卦六爻全动看用九，坤卦六爻全动看用六

### 5.5 六爻全静（无动爻）

- 以用神本身旺衰和日月对用神的生克为主
- 参考世应关系
- 卦伏吟之象，事情停滞

## 六、应期判断

应期是指所占之事应验的时间。

### 6.1 应期基本规则

| 用神状态 | 应期 |
|---|---|
| 用神旺相 | 逢合之日/月应（旺不待生，逢合则应） |
| 用神休囚 | 逢生旺之日/月应 |
| 用神受克 | 忌神受制之日应，或原神旺动之日应 |
| 用神旬空 | 出空之日应（冲空实空之日） |
| 用神入墓 | 冲墓之日应 |
| 用神被合 | 冲开之日应 |
| 用神月破 | 实破则无望；填实之日或合日可应 |
| 用神化进神 | 化进之支逢值或逢合之日应 |
| 用神化退神 | 化退之支逢值之日应（但事退缩） |

### 6.2 应期远近判断

- **近应看日** — 事情紧急或卦象显示近期的，以日来算
- **远应看月** — 事情非急或卦象显示较远的，以月来算
- **更远看年** — 极少数情况以年来算

### 6.3 应期综合判断

应期判断需要综合多个因素，不可拘泥于单一规则。一般原则：
- 用神有力，应期近
- 用神无力，应期远或不应
- 动爻多，变化快，应期近
- 静卦，事缓，应期远

## 七、分类占断简要规则

### 7.1 占财运

- **用神**: 妻财
- 妻财旺相得日月生扶 → 财运好，可得财
- 妻财休囚或空破 → 财运差，求财困难
- 子孙为财之原神（子孙生妻财），子孙旺相财源不断
- 兄弟为劫财之神（兄弟克妻财），兄弟旺动则破财

### 7.2 占事业（求职/升迁）

- **用神**: 官鬼
- 官鬼旺相 → 事业顺利，可成
- 官鬼持世 → 已有职位或职位与自身有缘
- 父母爻（印）旺相 → 有文凭/证书/上级支持
- 子孙动 → 克官鬼，不利事业（子孙为退隐之神）

### 7.3 占考试

- **用神**: 父母
- 父母旺相 → 考试顺利
- 官鬼为父母之原神，官鬼旺相有助考试
- 妻财动 → 克父母，不利考试
- 世爻旺相 → 自身状态好

### 7.4 占婚姻

- **男占用神**: 妻财；**女占用神**: 官鬼
- 世应相生相合 → 双方有缘
- 世应相冲相克 → 双方不合
- 用神旺相 → 对方条件好
- 六合卦 → 成婚，美满
- 六冲卦 → 难成或不久

### 7.5 占疾病

- **用神**: 官鬼（官鬼代表病）
- 官鬼旺相 → 病重
- 官鬼休囚 → 病轻或将愈
- 子孙（药神）旺相动 → 克制官鬼，病可愈
- 子孙空破 → 药不对症或无良医
- 用神（病人代表爻）旺相 → 体质尚好
- 世爻入墓/空亡 → 情况不乐观

### 7.6 占出行

- **用神**: 世爻
- 世爻旺相 → 出行顺利
- 世爻休囚受克 → 出行不利
- 应爻为目的地，应生世 → 目的地对我有利
- 官鬼动 → 途中有阻碍或灾祸
- 父母爻 → 代表车船等交通工具

### 7.7 占失物

- **用神**: 妻财
- 妻财旺相不空 → 可找回
- 妻财空亡/入墓 → 难以找回
- 妻财在内卦 → 在家附近
- 妻财在外卦 → 在外面
- 应爻/玄武 → 与偷盗者有关
- 六合卦 → 物在隐蔽处
- 六冲卦 → 物已散失

## 八、断卦注意事项

1. **月建日辰最重要** — 不论卦中如何，月建日辰对爻的旺衰起决定性作用
2. **用神是核心** — 始终围绕用神分析，不可偏离
3. **动静有别** — 动爻能作用于他爻，静爻不能（除非暗动）
4. **变爻只作用于本位** — 变爻只回头生克本位动爻，不直接作用于其他爻
5. **空亡与月破需区分** — 旺空可用，衰空真空；月破无力
6. **综合分析** — 不可只看一个因素就下结论，需综合月建、日辰、动爻、空亡等多方面
7. **灵活变通** — 规则是死的，断卦需要灵活运用，结合实际情况
