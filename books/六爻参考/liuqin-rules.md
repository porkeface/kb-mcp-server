# 六亲配置规则

## 一、六亲定义

六亲是以本卦所属宫的五行为"我"，根据每爻地支的五行与"我"的生克关系来确定。

| 关系 | 六亲名称 | 说明 |
|---|---|---|
| 同我者 | **兄弟** | 与宫五行相同 |
| 生我者 | **父母** | 该爻五行生宫五行 |
| 我生者 | **子孙** | 宫五行生该爻五行 |
| 克我者 | **官鬼** | 该爻五行克宫五行 |
| 我克者 | **妻财** | 宫五行克该爻五行 |

## 二、五行×五行 → 六亲对照表

以宫五行（我）为行，爻地支五行为列：

| 宫(我)＼爻 | 金 | 木 | 水 | 火 | 土 |
|---|---|---|---|---|---|
| **金宫** | 兄弟 | 妻财 | 子孙 | 官鬼 | 父母 |
| **木宫** | 官鬼 | 兄弟 | 父母 | 子孙 | 妻财 |
| **水宫** | 父母 | 子孙 | 兄弟 | 妻财 | 官鬼 |
| **火宫** | 妻财 | 父母 | 官鬼 | 兄弟 | 子孙 |
| **土宫** | 子孙 | 官鬼 | 妻财 | 父母 | 兄弟 |

**使用方法**: 先确定本卦所属宫（查 shiyao-table.md），取该宫五行作为"我"，再对每爻的地支查五行（查 najia-table.md 地支五行表），在上表中交叉查出六亲。

## 三、变卦六亲说明

- 变卦的六亲仍然以**本卦所属宫的五行**为"我"来配
- 不以变卦自身所属宫的五行为准
- 这是因为占卜的主体（宫）始终以本卦为准

## 四、用神选取表

根据占问事项选取用神（即卦中代表所问之事的那一爻）：

| 占问事项 | 用神（六亲） | 说明 |
|---|---|---|
| **求财、生意、收入** | 妻财 | 我克者为财 |
| **事业、功名、升职** | 官鬼 | 克我者为官 |
| **考试、文书、合同、房屋** | 父母 | 生我者为父母，主文书 |
| **婚姻（男问）** | 妻财 | 男以财为妻 |
| **婚姻（女问）** | 官鬼 | 女以官为夫 |
| **子女、下属、学生** | 子孙 | 我生者为子孙 |
| **兄弟、朋友、同事** | 兄弟 | 同我者为兄弟 |
| **疾病（自占）** | 官鬼 | 官鬼为病，看其旺衰 |
| **疾病（占长辈）** | 父母 | 用神为父母 |
| **出行** | 世爻 | 世爻代表自己，看世爻旺衰 |
| **失物** | 妻财 | 财为失物 |
| **官司（原告）** | 世爻 | 世为我，应为对方 |
| **官司（被告）** | 世爻 | 世为我，官鬼为案情 |
| **天气** | 子孙(晴)/父母(雨) | 子孙主晴，父母主雨 |
| **求职、求官** | 官鬼 | 官鬼为职位 |
| **投资、股票** | 妻财 | 财为收益 |
| **租房、买房** | 父母 | 父母主房屋 |
| **旅行平安** | 世爻 | 世爻代表自身安全 |

## 五、用神不现的处理

如果卦中六爻没有所需的用神六亲，则需要找**伏神**。详见 `duangua-advanced.md` 中飞神伏神章节。

简要规则：
- 在本卦所属宫的**本宫卦**（八纯卦）中找到该六亲所在的爻位
- 该爻称为"伏神"，其上方对应的本卦之爻称为"飞神"
- 伏神若得飞神生扶或日月生扶，则有用
- 伏神若被飞神克制或休囚无气，则无力
