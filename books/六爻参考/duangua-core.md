# 断卦核心规则 — 《增删卜易》

## 一、用神体系

### 1.1 四神定义

| 名称 | 定义 | 作用 |
|---|---|---|
| **用神** | 代表所问之事的爻（根据六亲选取） | 断卦核心，判断吉凶的主要依据 |
| **原神** | 生用神的爻（按五行生克） | 用神的保护者，原神旺则用神有源 |
| **忌神** | 克用神的爻（按五行生克） | 用神的破坏者，忌神旺则用神受损 |
| **仇神** | 生忌神的爻（克原神的爻） | 间接损害用神 |

### 1.2 五行生克链

用神的五行确定后，四神关系如下：

| 用神五行 | 原神(生用神) | 忌神(克用神) | 仇神(生忌神) |
|---|---|---|---|
| 金 | 土 | 火 | 木 |
| 木 | 水 | 金 | 土 |
| 水 | 金 | 土 | 火 |
| 火 | 木 | 水 | 金 |
| 土 | 火 | 木 | 水 |

### 1.3 用神旺衰判断优先级

1. 月建对用神的影响（最重要）
2. 日辰对用神的影响（次重要）
3. 动爻对用神的影响
4. 变爻回头生克

## 二、月建影响

月建是断卦中最重要的外部因素，决定爻的旺相休囚。

### 2.1 旺相休囚死（按月建地支所属季节）

| 月建地支 | 所属季节 | 旺 | 相 | 休 | 囚 | 死 |
|---|---|---|---|---|---|---|
| 寅、卯 | 春 | 木 | 火 | 水 | 金 | 土 |
| 巳、午 | 夏 | 火 | 土 | 木 | 水 | 金 |
| 申、酉 | 秋 | 金 | 水 | 土 | 火 | 木 |
| 亥、子 | 冬 | 水 | 木 | 金 | 土 | 火 |
| 辰、戌、丑、未 | 四季 | 土 | 金 | 火 | 木 | 水 |

**说明**:
- **旺**: 当令，力量最强
- **相**: 受当令之气生扶，力量次强
- **休**: 生当令之气而泄气，力量平
- **囚**: 克当令之气而受制，力量弱
- **死**: 被当令之气所克，力量最弱

### 2.2 月建其他作用

| 作用 | 说明 |
|---|---|
| **月生** | 月建五行生爻之五行，该爻得月建之生，有力 |
| **月克** | 月建五行克爻之五行，该爻受月建之克，受伤 |
| **月合** | 爻之地支与月建地支成六合关系，爻被合住，暂不能动（但旺相之爻合而不碍） |
| **月破** | 爻之地支与月建地支成六冲关系，爻被月破，无力（真破，出月不破除非日辰助） |

## 三、日辰影响

日辰是断卦中第二重要的外部因素。

### 3.1 日辰作用一览

| 作用 | 条件 | 说明 |
|---|---|---|
| **日生** | 日支五行生爻之五行 | 爻得日辰之生，增力 |
| **日克** | 日支五行克爻之五行 | 爻受日辰之克，减力 |
| **日合** | 日支与爻地支成六合 | 爻被合住（合起/合绊视旺衰而定） |
| **日冲** | 日支与爻地支成六冲 | 旺相之爻被冲为"暗动"，休囚之爻被冲为"日破" |
| **暗动** | 日冲旺相静爻 | 该静爻视同动爻，可以生克他爻 |
| **日破** | 日冲休囚静爻 | 该爻如同月破，彻底无力 |

### 3.2 暗动详解

- 静爻本身不能生克他爻
- 但如果静爻被日辰冲动（且该爻旺相），则该爻"暗动"
- 暗动之爻可以生克他爻，视同动爻
- 休囚之爻被日冲则为"日破"，不作暗动论

## 四、动爻规则

### 4.1 基本规则

| 规则 | 说明 |
|---|---|
| 动爻可以生克他爻 | 包括生克用神、原神、忌神等 |
| 静爻不能生克他爻 | 除非被日冲暗动 |
| 动爻之间可以互相生克 | 多爻同动时互相作用 |
| 动爻化出的变爻 | 只对本位动爻有回头生克的作用 |

### 4.2 变爻回头生克

动爻变出的变爻，对本位的动爻有以下作用：

| 类型 | 条件 | 效果 |
|---|---|---|
| **回头生** | 变爻五行生动爻五行 | 动爻得力，如虎添翼 |
| **回头克** | 变爻五行克动爻五行 | 动爻受制，力量大减，凶 |
| **回头合** | 变爻地支与动爻地支六合 | 动爻被合住，不能发挥作用 |
| **回头冲** | 变爻地支与动爻地支六冲 | 动爻受冲，不稳定 |

### 4.3 动爻生克优先级

1. 变爻先作用于本位动爻（回头生克）
2. 然后动爻再作用于其他爻
3. 如果动爻被变爻回头克，则该动爻无力生克他爻

## 五、空亡（旬空）

### 5.1 空亡判断

根据日柱所在旬确定空亡的两个地支（见 SKILL.md 旬空速查表）。卦中地支为空亡地支的爻即为"旬空"。

### 5.2 空亡分类

| 类型 | 条件 | 效果 |
|---|---|---|
| **旺相空** | 爻在月建旺相而旬空 | 空而不空，出空（过了该旬）即可发挥作用，称"有气之空" |
| **休囚空** | 爻在月建休囚而旬空 | 真空，无力，即使出空也难以恢复 |
| **动爻空** | 动爻旬空 | 动而逢空，事有变卦，冲空之日或出空之时应验 |
| **用神空** | 用神旬空 | 所问之事目前无着落，需等出空 |
| **忌神空** | 忌神旬空 | 忌神无力克害用神，反为好事 |

### 5.3 空亡要诀

- **空逢冲则实** — 空亡之爻被日辰冲，冲空则实，不再为空
- **空逢合则不空** — 需视情况而定，有时合住更不好
- **月破逢空** — 真空真破，完全无力
- **动爻化空** — 事情先有后无，或有阻碍

- **冲**: 冲则散，冲则动。旺爻逢冲为暗动，衰爻逢冲为破

## 七、入墓

### 7.1 五行入墓表

| 爻之五行 | 墓地（地支） |
|---|---|
| 金 | 丑 |
| 木 | 未 |
| 水 | 辰 |
| 火 | 戌 |
| 土 | 辰 |

### 7.2 入墓类型

| 类型 | 说明 |
|---|---|
| **入日墓** | 爻之五行入日辰之墓，如金爻遇丑日 |
| **入月墓** | 爻之五行入月建之墓 |
| **入动墓** | 爻之五行入动爻之墓地 |
| **化入墓** | 动爻变出的变爻为该动爻的墓地 |

### 7.3 入墓效果

- 入墓如入牢笼，爻被困住无法发挥作用
- 用神入墓主事情受阻、不顺
- 冲墓之日可解墓（如金入丑墓，未日冲丑可解）

## 八、断卦综合流程

1. **确定用神** → 根据所问之事在 liuqin-rules.md 用神选取表中找到对应六亲
2. **找到用神在卦中的位置** → 若不在卦中，找伏神（见 duangua-advanced.md）
3. **判断用神旺衰** → 月建决定旺相休囚死
4. **看日辰** → 日生/日克/日合/日冲/日破
5. **看空亡** → 用神是否旬空
6. **看入墓** → 用神是否入墓
7. **看动爻** → 动爻对用神的生克、变爻回头生克
8. **看原神** → 原神是否有力（旺相、动、得日月生扶）
9. **看忌神** → 忌神是否受制（休囚、空、被克制）
10. **综合判断** → 用神旺相有力则吉，休囚无力则凶

**吉凶判断要诀**:
- 用神旺相 + 得日月生扶 + 原神有力 → **大吉**
- 用神旺相 + 无忌神克害 → **吉**
- 用神休囚 + 得日月生扶或原神动来生 → **先凶后吉**或**有救**
- 用神休囚 + 无生扶 + 忌神旺动 → **凶**
- 用神旬空/入墓/月破 → 需细看是否有解救
