# 六神规则

## 一、六神顺序

六神固定顺序为：**青龙 → 朱雀 → 勾陈 → 螣蛇 → 白虎 → 玄武**

从初爻到上爻依次排列，初爻起哪一神由日干决定。

## 二、日干 → 初爻起神对照表

| 日干 | 初爻起神 |
|---|---|
| 甲、乙 | 青龙 |
| 丙、丁 | 朱雀 |
| 戊 | 勾陈 |
| 己 | 螣蛇 |
| 庚、辛 | 白虎 |
| 壬、癸 | 玄武 |

## 三、排列示例

若日干为**丙**或**丁**，初爻起**朱雀**：

| 爻位 | 六神 |
|---|---|
| 上爻 | 青龙 |
| 五爻 | 玄武 |
| 四爻 | 白虎 |
| 三爻 | 螣蛇 |
| 二爻 | 勾陈 |
| 初爻 | 朱雀 |

排列方法：从初爻起始神开始，按固定顺序（青龙→朱雀→勾陈→螣蛇→白虎→玄武）循环排列至上爻。

## 四、六神含义

六神在断卦中提供辅助信息，帮助判断事物的性质和特征：

| 六神 | 五行 | 基本含义 | 正面意象 | 负面意象 |
|---|---|---|---|---|
| **青龙** | 木 | 喜庆、吉祥 | 婚嫁喜事、财富、贵人 | 酒色过度、沉迷享乐 |
| **朱雀** | 火 | 口舌、文书 | 文采、消息、书信、考试 | 口舌是非、争吵、官讼 |
| **勾陈** | 土 | 迟滞、田土 | 房产、土地、稳定 | 拖延迟缓、牵绊、忧虑 |
| **螣蛇** | 土 | 虚惊、怪异 | 机变灵活 | 虚惊怪梦、缠绕不清、欺骗 |
| **白虎** | 金 | 凶险、刚烈 | 权威、军警、果断 | 血光、疾病、丧事、伤灾 |
| **玄武** | 水 | 暗昧、盗贼 | 智慧、谋略 | 偷盗、欺诈、暧昧、隐私 |

## 五、六神在断卦中的应用

- 六神主要起**辅助判断**作用，不作为主要吉凶依据
- 当爻象吉凶不明时，六神可提供方向性参考
- 六神可帮助判断事情的**性质**（如白虎临官鬼可能涉及疾病伤灾）
- 六神与六亲结合分析效果更佳：
  - 青龙临妻财：正财、喜财
  - 朱雀临父母：文书、考试
  - 白虎临官鬼：疾病、灾祸
  - 玄武临妻财：暗财、不正当之财
  - 勾陈临兄弟：拖延、争夺
  - 螣蛇临父母：虚假文书、怪梦
